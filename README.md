## 📌 Overview

RecipeHub is a modern recipe web application where users can search, explore,
and view detailed cooking recipes in a clean and responsive interface.

It provides fast recipe browsing with a smooth user experience and
well-structured recipe details.

---

## 🚀 Features

- 🔍 Recipe search by name or keyword
- 🍛 Recipe listing page with multiple recipes
- 📖 Detailed recipe page (ingredients + steps)
- ⚡ Fast and responsive UI (mobile + desktop)
- 🎨 Modern UI using Tailwind CSS
- 🌐 Deployed on Vercel
- 🧭 Smooth page navigation

---

## 🛠️ Tech Stack

- Next.js / React.js
- Tailwind CSS
- Recipe API (External Data Source)
- Vercel (Deployment)
- Git & GitHub -Node.js -MongoDB -Express.js -JWT -Butter_Auth
