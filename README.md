A modern full-stack recipe sharing platform built with the latest web
technologies. Users can discover new recipes, share their own creations, and
enjoy a seamless and interactive user experience.

✨ Key Features

- 🔐 **Secure Authentication:** Secure user registration and login system using
  **Better Auth** and **JSON Web Token (JWT)**.
- 🎨 **Dark & Light Mode:** Global theme switching with **Next Themes**,
  allowing users to switch between dark and light modes without losing their
  preferred theme during navigation.
- 🍳 **Browse & Search Recipes:** Easily explore and search thousands of
  delicious recipes with a smooth browsing experience.
- 📝 **Add Custom Recipes:** Users can create, publish, and share their own
  recipes with the community.
- 📊 **User & Admin Dashboard:** Separate dashboards for users and
  administrators to manage recipes, profiles, and platform content efficiently.
- 💳 **Stripe Payment Integration:** Secure payment gateway powered by
  **Stripe** for premium features or donations.
- 💫 **Smooth Animations:** Beautiful page transitions and micro-interactions
  implemented using **Framer Motion** for a modern user experience.

# 🎯 User Experience (UX)

- **Modern UI:** Clean, responsive, and visually appealing interface built with
  HeroUI and Tailwind CSS.
- **Fast Performance:** Optimized with Next.js Server-Side Rendering (SSR) and
  modern optimization techniques for faster page loading.
- **Interactive Experience:** Smooth animations and page transitions powered by
  Framer Motion.
- **Easy Navigation:** Simple and intuitive navigation between the dashboard,
  recipe pages, and recipe details.
- **Theme Support:** Users can switch between Dark and Light Mode based on their
  preference for a comfortable viewing experience.
